## 简介

BLFoundation是一套针对于Foundation的扩展.

## 使用

- 使用CocoaPods安装

  ```
  pod 'BLFoundation'
  or pod 'BLFoundation/Data'
  or pod 'BLFoundation/Date'
  or pod 'BLFoundation/Number'
  or pod 'BLFoundation/Data'
  or pod 'BLFoundation/Tools'
  ```

- 导入

  - swift

    ```swift
    import BLFoundation
    ```

  - objc

    ```objective-c
    #import<BLFoundation/BLFoundation.h>
    ```
